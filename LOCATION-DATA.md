## Tips for Writing Unique Content

### DO:
- Mention specific neighborhoods/areas in each location
- Include local hospital names
- Reference local landmarks or characteristics
- Use real delivery times based on actual routes
- Include population stats when available

### DON'T:
- Copy-paste generic content
- Use the same testimonials across locations
- Make up fake statistics
- Use identical hospital lists for nearby cities

### Content Length Guidelines:
- Intro paragraphs: 50-80 words each (100-160 total)
- Testimonials: 20-40 words each
- FAQ answers: 30-50 words each
