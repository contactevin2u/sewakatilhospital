## 3. Technical SEO Checklist <a name="technical-seo-checklist"></a>

### Essential Technical Requirements

#### Website Performance
- [ ] **Page Speed:** Target < 3 seconds load time
- [ ] **Core Web Vitals:**
  - LCP (Largest Contentful Paint): < 2.5s
  - FID (First Input Delay): < 100ms
  - CLS (Cumulative Layout Shift): < 0.1
- [ ] **Image Optimization:**
  - Use WebP format
  - Implement lazy loading
  - Serve responsive images (srcset)
  - Compress all images (< 200KB for hero images)

#### Mobile Optimization
- [ ] Responsive design (mobile-first approach)
- [ ] Touch-friendly buttons (min 44x44px)
- [ ] Readable text without zooming (16px+ base font)
- [ ] No horizontal scrolling
- [ ] Mobile menu functionality

#### Security & SSL
- [ ] SSL certificate installed (HTTPS)
- [ ] Force HTTPS redirect
- [ ] HSTS header enabled
- [ ] Mixed content warnings resolved

#### Crawlability & Indexing
- [ ] robots.txt configured correctly
- [ ] XML sitemap submitted to Google Search Console
- [ ] Canonical tags on all pages
- [ ] No orphan pages
- [ ] Breadcrumb navigation on all pages
- [ ] Custom 404 error page

#### URL Structure
- [ ] Clean, descriptive URLs
- [ ] Lowercase letters only
- [ ] Hyphens between words
- [ ] No parameters in URLs (where possible)
- [ ] Max 3 directory levels

**URL Examples:**
```
GOOD:
/products/portable-oxygen-concentrator/
/locations/kuala-lumpur/
/blog/cara-guna-mesin-oksigen-di-rumah/

BAD:
/products/product.php?id=123
/loc/kl/
/blog/post/1234/
```

#### Structured Data
- [ ] Organization schema on all pages
- [ ] LocalBusiness schema on location pages
- [ ] Product schema on product pages
- [ ] FAQ schema on FAQ and relevant pages
- [ ] Article schema on blog posts
- [ ] BreadcrumbList schema on all pages
- [ ] Review/Rating schema on testimonials

### Technical Implementation Checklist

```html
<!-- Essential Meta Tags (every page) -->
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="robots" content="index, follow">
<link rel="canonical" href="https://oxygencare.my/current-page/">

<!-- Preconnect for performance -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://www.googletagmanager.com">

<!-- Defer non-critical JS -->
<script src="/js/main.js" defer></script>

<!-- Async analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_ID"></script>
```

## 4. On-Page SEO Guidelines <a name="on-page-seo-guidelines"></a>

### Title Tag Formula
```
[Primary Keyword] | [Secondary Keyword] | Brand Name
Maximum 60 characters
```

**Examples:**
- `Sewa Mesin Oksigen Malaysia | Portable Oxygen Concentrator Rental & Sale`
- `Portable Oxygen Concentrator Malaysia | Sewa Mulai RM150/Minggu`
- `Sewa Mesin Oksigen Kuala Lumpur | Penghantaran Percuma KL`

### Meta Description Formula
```
[Primary keyword naturally included] + [Value proposition] + [CTA] + [Phone number if space]
Maximum 155-160 characters
```

**Examples:**
- `Sewa atau beli mesin oksigen mudah alih di Malaysia. Penghantaran percuma ke seluruh Semenanjung. Harga berpatutan mulai RM150/minggu. Hubungi 03-7890 1234.`

### Heading Structure (Every Page)

```html
<h1>One H1 per page - contains primary keyword</h1>

<h2>Main section headings - contain secondary keywords</h2>
    <h3>Subsection headings</h3>
        <h4>If needed for deeper content</h4>

<h2>Another main section</h2>
    <h3>Subsection</h3>
```

### Content Guidelines

1. **Keyword Density:** 1-2% for primary keyword, naturally distributed
2. **Content Length:**
   - Product pages: 800-1500 words
   - Location pages: 1000-1500 words
   - Blog posts: 1500-2500 words
   - Category pages: 500-800 words

3. **Content Elements:**
   - Use bullet points and numbered lists
   - Include tables for comparisons
   - Add relevant images every 300-400 words
   - Include internal links (3-5 per page minimum)
   - Add external links to authoritative sources (when relevant)

### Image SEO

```html
<!-- Image with proper SEO attributes -->
<img
    src="/images/products/philips-simplygo-mini.webp"
    alt="Philips SimplyGo Mini portable oxygen concentrator untuk travel"
    width="400"
    height="400"
    loading="lazy"
>
```

**Alt Text Formula:**
`[Product/Subject] + [Descriptor] + [Location if relevant]`

**Examples:**
- `Portable oxygen concentrator Philips SimplyGo Mini untuk travel`
- `Penghantaran mesin oksigen di Kuala Lumpur`
- `Pesakit warga emas menggunakan home oxygen concentrator`

## 12. Performance Optimization <a name="performance"></a>

### Page Speed Optimization

1. **Image Optimization:**
   ```html
   <!-- Use WebP with fallback -->
   <picture>
       <source srcset="image.webp" type="image/webp">
       <img src="image.jpg" alt="Description">
   </picture>

   <!-- Lazy loading -->
   <img loading="lazy" src="image.webp" alt="Description">
   ```

2. **CSS Optimization:**
   - Minify CSS files
   - Inline critical CSS
   - Remove unused CSS
   - Use CSS custom properties

3. **JavaScript Optimization:**
   - Defer non-critical JS
   - Minify and bundle
   - Remove unused code
   - Use async for analytics

4. **Server Optimization:**
   - Enable GZIP compression
   - Browser caching (1 year for static assets)
   - CDN for static files
   - HTTP/2 enabled

### Recommended Tools

- **Speed Testing:** Google PageSpeed Insights, GTmetrix, WebPageTest
- **Image Compression:** TinyPNG, Squoosh
- **CSS/JS Minification:** Terser, cssnano
- **CDN:** Cloudflare, AWS CloudFront
